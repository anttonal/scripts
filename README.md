# scripts

- `timer`: minimal terminal countdown timer; default input is seconds, `-m` treats input as minutes
- `open-daily-note`: toggle today's daily note in Neovim, creating it from the template and carrying over yesterday's `Tomorrow` items into the `Today` section; also supports legacy `TODO` headings
- `sysmaintenance.sh`: system maintenance
- `windows-reboot`: reboot to Windows
- `hypr-toggle-opacity`: toggle active Hyprland window opacity between 0.8 and 1.0
- `pick-wallpaper`: rofi picker to select wallpaper from `~/pictures/wallpaper/favorites` and apply it to one or all monitors via hyprpaper
- `rofi-toggle.sh`: toggle rofi app launcher; kills rofi if running, otherwise opens drun
- `toggle-monitor`: toggle secondary monitor DP-3 (LG ULTRAGEAR) on/off via hyprctl
- `volume-hold.sh`: thin wrapper around pamixer, passes all arguments through
